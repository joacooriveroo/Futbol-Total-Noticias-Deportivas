# Fútbol Total - Sitio Web de Noticias Deportivas

Estructura base y esqueleto semántico de la página de inicio y secciones secundarias de un portal de fútbol y actualidad deportiva, desarrollado con HTML5 puro.

## Estructura del Proyecto

* **`index.html`**: Página principal del sitio web ubicada en la raíz.
* **`pages/`**: Carpeta que almacena los archivos HTML secundarios:
  * `inicio.html`
  * `sobre-mi.html`
  * `proyectos.html`
  * `contacto.html`
* **`assets/`**: Carpeta que contiene los recursos multimedia e imágenes del sitio.

## Características

* Uso estricto de etiquetas semánticas de HTML5 (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<figure>`, `<footer>`).
* Navegación interconectada mediante rutas relativas.
* Imágenes optimizadas con atributos `alt` descriptivos y elementos `<figcaption>`.
